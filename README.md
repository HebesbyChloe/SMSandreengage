
  # SMS Chat Interface Design

  This is a code bundle for SMS Chat Interface Design. The original project is available at https://www.figma.com/design/wM0mDUWgEH73WlunoSOEbk/SMS-Chat-Interface-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deployment

  This project is configured for deployment on Vercel. See [VERCEL_SETUP.md](./VERCEL_SETUP.md) for detailed setup instructions.

  Quick start:
  1. Install the Vercel extension in VS Code/Cursor
  2. Sign in to Vercel via the extension
  3. Deploy using "Vercel: Deploy Project" command
  4. Configure environment variables in Vercel dashboard
  5. Update Twilio webhook URL to point to your deployment
